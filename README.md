# flutter_app_widgets_image

## Fonctionnalités

- **AppBar** avec titre centré et couleur bleue
- **Message de bienvenue** stylisé en orange
- **Image d'ambiance** chargée depuis une URL
- **Description** du café sur une seule ligne centrée
- **Boutons interactifs** "Commencer" et "En savoir plus"

## Widgets Utilisés

- `AppBar` - Navigation
- `Container` - Layout principal  
- `Column` - Disposition verticale
- `Row` - Boutons horizontaux
- `Image.network` - Image externe
- `ElevatedButton` - Bouton principal
- `OutlinedButton` - Bouton secondaire
- `SizedBox` - Espacements
